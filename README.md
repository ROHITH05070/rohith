📊 Daily Habit Tracker & Productivity Analysis System
📌 Project Overview

This project is a data-driven habit tracking and productivity analysis system built using an Excel dataset.
It tracks daily habits, goals, and weekly completion status (Monday to Sunday) to help users analyze discipline and consistency.

🎯 Objective

To build a system that:

Tracks daily habits
Monitors goal completion
Stores weekly progress (Mon–Sun)
Helps analyze productivity and discipline patterns
📁 Dataset Description

The dataset is stored in Excel format and contains:

Daily Habits → List of habits (e.g., Gym, Meditation, Study)
Goal → Target value for each habit
Mon–Sun Columns → Weekly tracking of habit completion
Index (#) → Habit identifier

Example habits:

Wake up early ⏰
Drink water 💧
Gym workout 🏋️‍♂️
Study time 🎓
Meditation 🧘
🧠 Key Features
Weekly habit tracking system
Goal-based productivity monitoring
Structured habit dataset
Easy to extend into dashboard or ML model
Clean tabular format for analysis

📊 How It Works
User defines daily habits
Each habit has a target goal
Weekly progress is recorded (Mon–Sun)
Data is analyzed to measure consistency and performance
